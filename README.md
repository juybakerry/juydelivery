# 🥐 จุ้ย เบเกอรี่ — แอปสั่งขนมออนไลน์

แอปพลิเคชันสั่งขนมออนไลน์สำหรับร้านจุ้ย เบเกอรี่ จังหวัดอำนาจเจริญ  
พัฒนาเป็น Progressive Web App (PWA) — ติดตั้งได้บนมือถือโดยไม่ต้อง Play Store

---

## 🚀 วิธีเปิดใช้งานออนไลน์ด้วย GitHub Pages

### ขั้นตอนที่ 1 — Upload ขึ้น GitHub
1. ไปที่ [github.com](https://github.com) → สร้าง Repository ใหม่
2. ตั้งชื่อ เช่น `juice-bakery` → กด **Create repository**
3. กด **uploading an existing file** แล้วลากไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้นไป:
   - `index.html`
   - `images.js`
   - `sw.js`
   - `manifest.json`
   - `icon-192.png`
   - `icon-512.png`
4. กด **Commit changes**

### ขั้นตอนที่ 2 — เปิด GitHub Pages
1. ไปที่ **Settings** → **Pages**
2. Source: เลือก **Deploy from a branch**
3. Branch: เลือก **main** → **/ (root)** → กด **Save**
4. รอ 1-2 นาที แอปจะพร้อมที่ URL:  
   `https://ชื่อuser.github.io/juice-bakery/`

## 📱 วิธีติดตั้งบนมือถือ (Android)
1. เปิด URL ด้วย Chrome
2. กด **เมนู 3 จุด** → **"เพิ่มในหน้าจอหลัก"** หรือ **"ติดตั้งแอป"**
3. แอปจะปรากฏบนหน้าจอเหมือน App ปกติ

## 📱 วิธีติดตั้งบนมือถือ (iPhone/iPad)
1. เปิด URL ด้วย Safari
2. กดปุ่ม **แชร์** (กล่องมีลูกศรขึ้น)
3. เลือก **"เพิ่มไปยังหน้าจอโฮม"**
